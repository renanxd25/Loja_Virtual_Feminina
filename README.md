# Loja_Virtual_Feminina
Front End de Loja virtual feminina. Desenvolvida com HTML5, CSS, SCSS e BOOTSTRAP. O site contem plugins do facebook com comentarios.
O site é uma representação de um Ecomerce de uma loja virtual de um site de roupas femininas.
